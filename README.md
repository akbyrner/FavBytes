Thanks for using FavBytes! This app is designed to let you save pictures of your favorite dishes, drinks, or other restaurant things, to a map. This solves your problem of googling the same descriptions of menu items 'near you' and allows you to just see the things FavBytes users actually like when deciding on a restaurant.

## **Features:**

### **The Map:**

The Map will show you your current location, and any saved bytes nearby.
You can scroll through the 5 closest locations to you in your collection.

### **The Gallery:**

The gallery allows you to quickly swipe through your FavBytes. This way you can see all of the items saved in order to select a Byte page

### **The Byte Page:**

This allows you to see the description, tags, and ratings attached to an individual byte.

### **The Upload Page:**

Select a restaurant location using search on the map or search using the 'not right location' button on the upload page. Then, save a picture, a description, a title, select tags, and choose an overall rating if you desire


## 🚀 Getting Started
### Prerequisites
- **Node.js**: v18.0.0 or higher
- **MongoDB**: A running instance (local or Atlas)
- **API Keys**: You will need credentials for Google OAuth, Mapbox, and AWS S3.
### Installation
1. Clone the repository:
   ```git clone https://github.com/Stack-mates/FavBytes```
   ```cd FavBytes```
2. Install dependencies
    ```npm install```
3. Set up your environment variables
    ```cp .env.example .env```
4. Run the program
    ```npm run start```
